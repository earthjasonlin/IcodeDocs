# Dev 机器人

说明：机器人（即角色）相关属性与方法

## 属性

### `x -> int`

**只读**，机器人的`x`坐标

### `y -> int`

**只读**，机器人的`y`坐标

## 方法

### `step(n) -> void`

机器人前进或后退指定步数，**本操作算作`n`步**

-   `n`: 整数，正为前进，负为后退

### `turnLeft() -> void`

机器人原地左转，**本操作算作`1`步**

### `turnRight() -> void`

机器人原地右转，**本操作算作`1`步**

## 示例用法

```python
# 获取机器人当前的x坐标
x = Dev.x
# 获取机器人当前的y坐标
y = Dev.y

# 机器人原地右转
Dev.turnRight()
# 机器人前进2步
Dev.step(2)
# 机器人原地左转
Dev.turnLeft()
# 机器人后退3步
Dev.step(-3)
```
